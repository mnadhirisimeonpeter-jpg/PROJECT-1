# PROJECT-1
Ace Tennis Academy Website
ICS2102 Web Application Development — Strathmore University
Mnadhiri Simeon Peter, 219809

This is a multi-page website for a fictional tennis academy based in Nairobi. Built as the semester project for ICS2102.

Pages: Home, About, Coaching, Schedule, Tournament, Gallery, Contact

Stack: HTML5, CSS3, JavaScript ES6+, Bootstrap 5.3, Google Fonts (Bebas Neue + Inter)

JavaScript features implemented:
- Contact form validation with real-time error messages
- Gallery filter by category (Training, Tournament, Fitness, Team)
- Live keyword search on the coaching programmes page
- Dark mode toggle that saves preference via localStorage
- Scroll reveal animations using IntersectionObserver

Colour palette: deep greens (#1B4332, #2D6A4F), gold accent (#E9C46A), dark navy (#0D1B2A).
